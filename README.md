## Setup

- Clone the repo
```bash
$ git clone git@github.com:TheNsBhasin/covid19india-flutter.git
$ cd covid19india-flutter/
```
- Checkout version
```dart
  flutter --version
```
- Run doctor
```dart
  flutter doctor
```
- Start app
```dart
  flutter pub get
  flutter run
```
